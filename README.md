# Loan-Risk-Analytics-Power-BI
Interactive Power BI dashboard for loan risk analysis, approval trends, credit score insights, and business intelligence reporting using financial data.
# 📊 Loan Risk Analytics Dashboard using Power BI

## 📌 Project Overview

This project transforms loan application data into meaningful business insights using **Power BI** and **Excel**. The dashboard helps financial analysts, risk managers, and decision-makers monitor loan performance, evaluate applicant risk levels, and identify trends in loan approvals and rejections.

The project demonstrates how Business Intelligence tools can support data-driven decision-making in financial institutions by providing real-time analytics and interactive visualizations.

---

# 🎯 Objectives

* Analyze loan application data
* Monitor loan approval and rejection trends
* Identify high-risk applicants
* Evaluate credit score distribution
* Support lending decisions through data visualization
* Demonstrate Business Intelligence in finance

---

# ⚙️ Dashboard Development Process

### Step 1 — Collect Loan Data

The dataset contains information related to:

* Applicant Income
* Loan Amount
* Credit Score
* Employment Status
* Loan Status
* Risk Category

---

### Step 2 — Import Data into Power BI

The Excel dataset is imported into Power BI Desktop using the Get Data feature.

---

### Step 3 — Clean and Transform Data

Data transformation includes:

* Handling missing values
* Formatting columns
* Creating calculated fields
* Categorizing risk levels

---

### Step 4 — Create DAX Measures

Key measures created:

* Total Applications
* Approved Loans
* Rejected Loans
* Approval Rate
* Average Credit Score

---

### Step 5 — Build Dashboard Visualizations

Visuals include:

* KPI Cards
* Pie Charts
* Bar Charts
* Risk Distribution Analysis
* Credit Score Trends

---

# 📊 Dashboard KPIs

| KPI                  | Description                          |
| -------------------- | ------------------------------------ |
| Total Applications   | Number of loan applications received |
| Approved Loans       | Loans approved                       |
| Rejected Loans       | Loans rejected                       |
| Approval Rate        | Percentage of approved applications  |
| Average Credit Score | Mean applicant credit score          |
| High Risk Applicants | Total high-risk customers            |
